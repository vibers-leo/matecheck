## 1. 현재 상태 진단

### 개발 완성도
- **코드 규모**: 16개 모델, 22개 마이그레이션, Rails 8.1 API-only 서버
- **주요 기능**: Nest(보금자리) 관리, 미션 시스템, 가계부, 일정, 정책 크롤러, RSVP 관리
- **프론트엔드**: React Native (Expo) + 웹뷰 (별도 개발)
- **인증**: bcrypt + has_secure_password (간단하지만 보안 강화 필요)
- **배포**: NCP Docker (포트 4010), PostgreSQL 프로덕션
- **운영**: 모든 코드가 OKR 중심으로 정렬됨

### 매출/사용자 현황 (추정)
- **상태**: 초기 성장 단계 (출시 전, 베타 테스트 진행)
- **대상 시장**: 공유 가구/룸메이트 (대학가 주변, 직장인 레지던스)
- **추정 시장규모**: 한국 공유 가구 약 300,000명
- **비즈니스 모델**: B2C 구독 (월 0-9,900원 범위, 아직 미정)

### 핵심 과제
1. **PMF 미검증**: 공유 생활 시뮬레이션 없이 기능만 먼저 구축된 상태
2. **경쟁 부재가 기회**: Stick(핀테크), NAVER 공동생활앱 등이 있지만 전문 앱 없음
3. **단위 경제 불명확**: 구독료 vs 거래 수수료 수익 모델 미결정
4. **사용자 확보 경로 없음**: "어떻게 첫 1,000명 모을까?" 전략 부재
5. **보안 강화**: 공금 관리(가계부) 기능이 있어 PCI-DSS 같은 규제 고려 필요

## 3. Beachhead 분석

### Primary Segment: 대학가 공동 임차 (쉐어하우스)

**4가지 선정 기준:**

1. **Segment Size**: 서울 주요 대학가(강남, 홍대, 신촌, 종로) 약 20,000명
2. **Execution Advantage**:
   - 고통이 명확함 (공금 정산 싸움, 청소 책임 불명)
   - 가격 민감도 낮음 (월 10만원 사이에서 안 줄어듦)
   - Word-of-mouth 강함 (대학 커뮤니티)
3. **Size of Beachhead**: 월 100-200개 Nest 온보딩 가능
4. **Growth Path**: 대학→기숙사(기업)→해외 유학생→가족 공동생활

### Go-to-Market

- **Phase 1 (0-2개월)**: 강남, 홍대 쉐어하우스 100개 핸드픽 + 초대
- **Phase 2 (2-4개월)**: 대학 캠퍼스 동아리 홍보 (포스터, 부스)
- **Phase 3 (4-6개월)**: Nest당 추천 인센티브 (친구 초대 시 1개월 무료)

## 4. Growth Loop

### Primary Loop: **Nest 규모 확대→미션 명확화→가계부 신뢰→더 많은 기능 사용→재계약**

```
초기 3명 Nest 가입
  ↓
미션 설정 (청소, 요리, 설거지)
  ↓
미션 완료율 90%+ (책임감 증가)
  ↓
공금 정산 투명화 (싸움 감소)
  ↓
신규 1-2명 추가 입주 (Nest 확대)
  ↓
기능 추가 사용 (캘린더, 규칙)
```

**Loop 시간**: 4-6주 (초기 설정→안정화)

### Secondary Loop: **초대→Word-of-Mouth→신규 Nest 창설**

- 기존 Nest의 친구가 "우리도 써야 해" → 새로운 Nest 생성
- 인센티브: 초대당 500포인트 (다음 달 결제 할인)

## 5. 가격 전략

### 추천 가격 모델 (3-Tier)

| 플랜 | 월가격 | Nest 규모 | 기능 | 타겟 |
|------|--------|---------|------|------|
| **Free** | 0원 | 1-3명 | 미션, 가계부 기본 | 학생(초기 체험) |
| **Pro** | 4,900원 | 3-8명 | 캘린더, 규칙, 고급 분석 | 대학가 쉐어하우스 |
| **Business** | 14,900원 | 8명+ | API 연동, 다중 Nest 관리, 우선지원 | 기업(기숙사, 임차) |

### 추가 수익원

#### 거래 수수료 (선택사항, Phase 2)
- 공금 정산 시 1% (실제 송금 시에만 청구)
- 예: 월 100,000원 정산 → 1,000원 수수료 (Pro 플랜에선 무료)

#### 프리미�ム 기능 (선택사항, Phase 3)
- 정책 분석: 월 2,900원 (공동생활 규칙 생성 AI)
- SMS 알림: 월 1,000원 (톡 외 문자 알림)

### 가격 근거
- 대학생 용돈: 월 30-50만원 → 월 5,000원은 1-2% (수용 가능)
- 기업 사원숙사: 월 임차료 100만원 → 월 15,000원은 1.5% (충분히 저가)
- 경쟁사 부재로 프리미엄 포지셔닝 가능

## 7. North Star Metric

### 추천 NSM: **월간 활성 Nest(MAA)당 평균 월 사용 시간**

- 목표: 4시간 (12개월 후)
- 초기 지표: 정산 정확도, 미션 완료율이 사용 빈도와 상관

### Input Metrics (주간 추적)

1. **Acquisition**:
   - 신규 Nest 가입 수 (목표: 주 30개)
   - 신규 사용자 수 (목표: 주 100명)
   - 가입 경로 분석 (캠퍼스 vs 입소문 vs 광고)

2. **Activation**:
   - 첫 주 미션 생성율 (목표: 70%)
   - 첫 주 공금 정산 사용율 (목표: 50%)
   - 초기 경험 NPS (목표: 50+)

3. **Engagement**:
   - 월간 활성 Nest (MAA)
   - Nest당 월간 미션 수 (목표: 12개, 주 3개)
   - 미션 완료율 (목표: 85%)
   - 공금 정산 월 거래액 (목표: Nest당 500,000원)

4. **Retention**:
   - 30일 재방문율 (목표: 60%)
   - 60일 재방문율 (목표: 40%)
   - Nest 존속율 (6개월: 70%)
