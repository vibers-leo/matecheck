## 🎨 디자인 가이드라인

### 레이아웃 구조

```
┌─────────────────────────┐
│                         │
│   헤드라인 (40%)        │  ← 큰 텍스트, 핵심 메시지
│                         │
├─────────────────────────┤
│                         │
│                         │
│   앱 화면 (50%)         │  ← 실제 앱 스크린샷
│                         │
│                         │
├─────────────────────────┤
│  부가 설명 (10%)        │  ← 작은 텍스트
└─────────────────────────┘
```

### 컬러 팔레트

```css
/* 브랜드 컬러 */
Primary: #FF7F50 (오렌지)
Secondary: #4A90E2 (블루)
Background: #F2F4F6 (라이트 그레이)
Text: #1A1A1A (다크 그레이)
White: #FFFFFF

/* 그라데이션 (선택) */
Gradient 1: #FF7F50 → #FF6B6B
Gradient 2: #4A90E2 → #64B5F6
```

### 타이포그래피

```css
/* 헤드라인 */
Font: Pretendard Bold / Apple SD Gothic Neo Bold
Size: 80-120px
Line Height: 1.2
Color: #1A1A1A 또는 #FFFFFF

/* 부제목 */
Font: Pretendard Medium
Size: 40-60px
Line Height: 1.4
Color: #666666

/* 설명 */
Font: Pretendard Regular
Size: 32-48px
Line Height: 1.5
Color: #999999
```

## 📱 스크린샷 구성 (4-6장 권장)

### 1번: 핵심 가치 제안 (Hero Shot)

**목적**: 첫인상으로 앱의 핵심 가치 전달

**배경**: 밝은 오렌지 그라데이션 (#FF7F50 → #FF6B6B)

**텍스트**:
```
큰 텍스트: "우리집이 행복해지는"
작은 텍스트: "공동 생활 관리 앱"
```

**앱 화면**: 홈 화면 (할 일 목록)

**시각 요소**:
- 집 아이콘 또는 하트 아이콘
- 행복한 가족 일러스트 (선택)

---

### 2번: 할 일 관리

**목적**: 핵심 기능 #1 강조

**배경**: 흰색 또는 라이트 그레이 (#F2F4F6)

**텍스트**:
```
큰 텍스트: "청소 당번 자동 배정"
작은 텍스트: "더 이상 싸우지 마세요"
```

**앱 화면**: 할 일 목록 화면
- 체크박스로 완료된 항목 보여주기
- 담당자 아바타 표시
- 반복 설정 아이콘 표시

**강조 요소**:
- 완료된 할 일에 체크 애니메이션
- 담당자 프로필 이미지

---

### 3번: 일정 공유

**목적**: 핵심 기능 #2 강조

**배경**: 블루 그라데이션 (#4A90E2 → #64B5F6)

**텍스트**:
```
큰 텍스트: "가족 일정 한눈에"
작은 텍스트: "일정 투표로 약속 잡기"
```

**앱 화면**: 캘린더 화면
- 여러 사람의 일정이 색깔별로 구분된 화면
- 투표 중인 일정 카드

**강조 요소**:
- 캘린더에 이벤트 표시
- 투표 진행 바

---

### 4번: 가계부

**목적**: 핵심 기능 #3 강조

**배경**: 그린 계열 (#2ECC71 → #27AE60)

**텍스트**:
```
큰 텍스트: "생활비 간편 관리"
작은 텍스트: "예산 대비 실지출 확인"
```

**앱 화면**: 가계부 화면
- 카테고리별 지출 그래프
- 이번 달 예산 vs 지출

**강조 요소**:
- 원형 차트 또는 막대 그래프
- 금액 숫자 강조

---

### 5번: 보금자리 전환 (선택)

**목적**: 차별화 기능 강조

**배경**: 보라색 계열 (#9B59B6 → #8E44AD)

**텍스트**:
```
큰 텍스트: "여러 보금자리 관리"
작은 텍스트: "가족, 회사, 동아리를 한 곳에"
```

**앱 화면**: 보금자리 전환 화면
- 여러 보금자리 카드
- 쉬운 전환 UI

## 🎯 모범 사례 (Best Practices)

### DO ✅

- **첫 번째 스크린샷이 가장 중요** (70% 사용자가 1번만 봄)
- **실제 앱 화면 사용** (신뢰도 향상)
- **큰 텍스트 사용** (모바일에서도 읽기 쉽게)
- **밝은 배경 우선** (앱스토어 흰색 배경과 조화)
- **일관된 디자인** (모든 스크린샷 스타일 통일)
- **현지화** (한국어 우선, 향후 영어 추가)

### DON'T ❌

- **텍스트 과다** (3줄 이상은 X)
- **복잡한 UI** (너무 많은 정보)
- **저해상도 이미지** (흐릿하거나 픽셀화)
- **거짓 정보** (실제 앱에 없는 기능 표시)
- **다크 배경 남용** (앱스토어에서 잘 안 보임)
- **너무 많은 스크린샷** (6장 이상은 오히려 역효과)
