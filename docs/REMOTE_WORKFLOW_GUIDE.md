| 도구 | 역할 | 주요 작업 |

| **Cowork (데스크탑)** | 프로젝트 매니저 | 문서 생성, 외부 서비스 연동, 자동화, 분석 |

## 3. Cowork 스케줄 태스크

| 태스크 ID | 스케줄 | 설명 |
|-----------|--------|------|
| `matecheck-daily-status` | 평일 오전 9시 | Git 변경사항, OKR 진행도, 에러 로그, 다음 할 일 요약 |

### 스케줄 태스크 복원 방법 (앱 재설치 시)

앱을 재설치하면 스케줄 태스크도 사라집니다. 아래 내용으로 Cowork에서 다시 생성하세요:

```
태스크 ID: matecheck-daily-status
설명: MateCheck 프로젝트 일일 상태 요약 리포트
스케줄: 0 9 * * 1-5 (평일 오전 9시)

프롬프트:
MateCheck(룸메체크) 프로젝트의 일일 상태를 요약해주세요.
프로젝트 폴더: Desktop/matecheck
GitHub: https://github.com/juuuno-coder/matecheck.git
1. Git 상태: 최근 커밋 내역(최근 3일), 현재 브랜치, 미커밋 변경사항
2. OKR 진행 상태: OKR.md의 Phase별 체크리스트 완료/미완료 요약
3. 주요 이슈: backend/error.log에 새로운 에러 확인
4. 다음 할 일 제안: OKR Initiatives 기반 우선 작업 2-3개 추천
```

## 4. GitHub 이슈 라벨 체계 (OKR 연동)

```
[KR1] 만족도    → 앱스토어 평점 관련
[KR2] 재방문율   → 성능, UX 개선
[KR3] 안정성    → 크래시, 에러 핸들링
[KR4] 성장      → 온보딩, 초대 기능
[Phase1]        → 배포 준비
[Phase2]        → QA
[Phase3]        → 런칭 준비
```
