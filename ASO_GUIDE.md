## 📸 5. 스크린샷 전략

### 필수 스크린샷 (최소 4장)

#### 1번: 핵심 가치 제안

```
[스크린샷]
큰 텍스트: "우리집이 행복해지는"
작은 텍스트: "공동 생활 관리 앱"
```

#### 2번: 할 일 관리

```
[스크린샷]
텍스트: "청소 당번 자동 배정"
부제: "더 이상 싸우지 마세요"
```

#### 3번: 일정 공유

```
[스크린샷]
텍스트: "가족 일정 한눈에"
부제: "일정 투표로 약속 잡기"
```

#### 4번: 가계부

```
[스크린샷]
텍스트: "생활비 간편 관리"
부제: "예산 대비 실지출 확인"
```

### 스크린샷 디자인 팁

- **상단 40%**: 핵심 메시지 (큰 텍스트)
- **중간 50%**: 실제 앱 화면
- **하단 10%**: 부가 설명 (작은 텍스트)
- **배경**: 밝은 그라데이션 또는 단색
- **폰트**: 굵고 읽기 쉬운 폰트

### 크기

- **iOS**: 1284 × 2778px (iPhone 14 Pro Max)
- **Android**: 1080 × 1920px (16:9 비율)

## 🎬 6. 프로모션 영상 (선택)

### iOS App Preview (30초)

**구성**:
1. **0-5초**: 문제 제시 ("청소 당번 또 싸우셨나요?")
2. **5-20초**: 솔루션 시연 (주요 기능 3가지)
3. **20-25초**: 결과 ("이제 우리집도 행복해요!")
4. **25-30초**: CTA ("지금 무료로 시작하세요")

## 📈 7. 평점 및 리뷰 관리

### 평점 목표: 4.5점 이상 (KR1)

**전략**:
1. **적절한 타이밍에 리뷰 요청**
   - 할 일 10개 완료 후
   - 일정 5개 등록 후
   - 앱 사용 7일 후

2. **좋은 경험 후 요청**
   - 성공 Toast 표시 직후
   - 목표 달성 시

3. **부정적 피드백 차단**
   - 앱 내 피드백 기능 먼저 제공
   - 문제 해결 후 리뷰 요청

### 리뷰 답변

```
긍정적 리뷰:
"소중한 리뷰 감사합니다! 앞으로도 더 나은 서비스로 보답하겠습니다 😊"

부정적 리뷰:
"불편을 드려 죄송합니다. 해당 문제는 다음 업데이트에서 개선될 예정입니다.
추가 문의는 support@matecheck.app로 연락 주세요."
```

## 📊 8. 성과 측정

### 주요 지표

| 지표 | 목표 | 측정 도구 |
|------|------|----------|
| 노출 횟수 | 10,000+ | App Store Connect / Play Console |
| 앱 페이지 방문 | 1,000+ | App Store Connect / Play Console |
| 설치 수 | 500+ | Analytics |
| 전환율 | 5%+ | (설치 수 / 방문 수) × 100 |

### A/B 테스트

- **아이콘**: 2-3가지 버전 테스트
- **스크린샷**: 순서 및 메시지 변경
